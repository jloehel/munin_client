Welcome to munin_client's documentation!
========================================

A simple munin client which uses pexpect.

Requirements
^^^^^^^^^^^^

* `pexpect` - Pexpect allows easy control of interactive console applications.
* `netcat` -  Netcat is a featured networking utility which reads and writes
  data across network connections, using the TCP/IP protocol.
